# Patcher-for-Eclipse-s-mod

A tool for patching an executable file to install the Eclipse's mod.

## Installation
- ### [Windows](Patcher.zip)
- ### [Linux](Patcher-x86_64.AppImage)

## Usage
- Select the executable file in the game files
- Select the patch file
- Сlick "Apply patch"
